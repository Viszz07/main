STEPS FOR CREATING JAR FILE 
1. CREATE DIRECTORY SRC/MAIN/JAVA/APP.JAVA  AND THE MAIN CLASSNAME AND FILENAME SHOULD MATCH
2. CREATE A BUILD.GRADLE FILE
3. ADD PLUGIN AND JAR MANIFEST ATTRIBUTES AND USE REPOSITORY AND DEPENCY ON WHEN THERE IS ONE 
4. IN TERMINAL GIVE COMMAND GRADLE BUILD
5. A JAR FILE WILL BE CREATED IN BUILD/LIBS/APP.JAR folDER
